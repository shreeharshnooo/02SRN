Student Portal prototype

This is a small Express + static frontend prototype for student course registration.

Contents:
- server.js — Express server and API
- public/ — static frontend
- data/ — JSON data store (sample data)

Quick start:
1. Install Node.js 18+ and Git
2. npm install
3. npm start

Note: Do not commit real user data. data/users.json is ignored by .gitignore.
